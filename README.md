# 🎁 GiftyAI – Smart Gift Recommendation Website

GiftyAI is a web application designed to help users find the perfect gift for their loved ones.
Instead of endlessly scrolling through ideas, users can simply enter preferences, budget, and interests — and get personalized gift suggestions instantly.

###LIVE DEMO LINK
https://vercel.com/anushkaverse10s-projects/gifty-ai/A8Fe6uXjDdK5CM3XYCxEdQ9geoE7
---

## 🚀 Features

* 🎯 Personalized gift recommendations
* 💸 Budget-based filtering
* ❤️ Interest-based suggestions (Gaming, Fitness, Cooking, etc.)
* 🤖 AI-powered chatbot (in progress)
* 🎨 Clean and modern UI
* ⚡ Fast and lightweight frontend

---

## 🛠️ Tech Stack

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/giftyai.git
cd giftyai
```

---

### 2. Setup backend

```bash
cd backend
npm install
node server.js
```

Server will run on:

```
http://localhost:5000
```

---

### 3. Run frontend

Open `index.html` using Live Server or browser.
